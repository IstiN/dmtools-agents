# Jira tracker comment

Use Jira wiki markup in `outputs/response.md`.

- Headings: `h1.`, `h2.`, `h3.`
- Bullets: `* item`
- Numbered lists: `# item`
- Bold: `*text*`
- Inline code: `{{code}}`
- Code block: `{code}...{code}`
- Link: `[title|url]`

Do not use Markdown headings, fenced code blocks, or backtick inline code.
